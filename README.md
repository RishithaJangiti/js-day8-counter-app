# Counter App

A simple Counter App built using JavaScript DOM manipulation.

## Features
- Increment counter
- Decrement counter
- Reset counter to zero
- Button click event handling

## Tech Stack
- HTML
- CSS
- JavaScript (DOM)

## How It Works
- Click **+** to increase the count
- Click **-** to decrease the count
- Click **Reset** to set the count back to 0

## Project Purpose
This project was built to practice:
- DOM manipulation
- Event listeners
- JavaScript logic

## Preview
Open `counterApp.html` in any browser to run the app.
